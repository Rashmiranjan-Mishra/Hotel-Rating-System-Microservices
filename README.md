# Hotel Rating System (Microservices)

A microservices-based application for managing hotel ratings. Includes services for:

- User Management
- Hotel Information
- Rating System
- API Gateway
- Config Server
- Service Registry

## Tech Stack
- Java / Spring Boot
- Eureka
- Spring Cloud Config
- JMeter (for performance testing)

## Getting Started
1. Clone the repo
2. Start Config Server → Eureka → All microservices
